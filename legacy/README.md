# Legacy Canvas 2D prototype (2017)

These files are the original TeEngine prototype:

- `main.js` — Canvas 2D render loop
- `particle.js` — Custom particle physics (springs, gravity)
- `utils.js` — Math/collision helpers

Superseded by the TypeScript + WebGPU engine in `src/`. Useful math from `utils.js` will be ported to `src/math/` over time.
